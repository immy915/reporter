# ログイン画面

## 部品

- ID(テキストボックス)：`user-id`
- パスワード(パスワードボックス)：`password`
- ログインボタン：`submit`
- パスワード再発行(リンク)

## 機能

### ログイン

- 未入力チェック
    - 未入力の場合メッセージを表示
        - 「未入力の項目があります」
- ID/パスワードが一致
    - ログイン前に表示していたページに遷移
- ID/パスワードが一致しない
    - メッセージを表示
        - 「IDもしくはパスワードが間違っています」
    - IDは入力値を保つ
