# 観劇レポ共有サイト（仮）

## 検討事項

- データベース設計
- 課金サーバーの要否
- 名前
- URL
- 規約

## 方針

- あくまで個人個人の記録
- ユーザー同士の交流はなし
- あまりにひどい誹謗中傷はNG
- 過剰な個人情報は登録させない
