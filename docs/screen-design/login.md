# ログイン画面

## 部品

- ID(テキストボックス)：`user_id`
    - 未入力チェック
- パスワード(パスワードボックス)：`password`
    - 未入力チェック
- ログインボタン：`submit`
- パスワード再発行(リンク)

## 機能

### ログイン

- 未入力チェック
    - 未入力の場合メッセージを表示
        - `empty_err`:「未入力の項目があります」
- ID/パスワードが一致
    - ログイン前に表示していたページに遷移
- ID/パスワードが一致しない
    - メッセージを表示
        - `login_err:`「IDもしくはパスワードが間違っています」
    - IDは入力値を保つ
- ログイン後はサイドメニューが各ページに表示される
